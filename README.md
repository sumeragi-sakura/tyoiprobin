# tyoiprobin

「Data Aggregation」をご利用いただきありがとうございます。

## 1. インストール方法

ご利用の環境や好みに合わせて、2パターンのいずれかで導入してください。

### A. インストーラーを使用する場合（推奨）
1. `Choipro_Setup_vX.X.X.exe` をダウンロードして実行します。
2. 画面の指示に従ってインストールを完了させてください。
3. デスクトップまたはスタートメニューに作成されたショートカットから起動できます。

### B. バイナリ（EXEファイル）を直接利用する場合
1. `ちょいプロ(仮).exe` を任意のフォルダにダウンロードします。
2. ダウンロードしたEXEファイルをダブルクリックして起動します。

**【設定データベース（.dbファイル）について】**
直接バイナリを実行する場合でも、設定を保存するデータベースファイル（`settings.db`）は、Windowsの標準的なデータ保存領域に**初回起動時に自動作成**されます。
*   **配置場所**: `%APPDATA%\Data Aggregation\settings.db`
    *   エクスプローラーのアドレスバーに上記パスを貼り付けてEnterを押すと確認できます。
    *   PCの買い替えなどで設定を引き継ぎたい場合は、このファイルを新しいPCの同じ場所にコピーしてください。

## 2. 機能の使用方法

### ① 撮影時刻分割
MMSの走行データ（CSV）から、電柱ごとの撮影時刻を抽出・整理します。
1. **出力ファイル名**: 保存するファイルの名前を入力します。
2. **走行車両名**: 記録する車両名（例：1号車）を入力します。
3. **元ファイル**: 「ファイル参照(一括)」から、元のCSVファイルを複数選択します。
4. **実行**: 「CSV出力」をクリックし、保存先フォルダを選択すると、整理されたCSVが作成されます。

### ② 画像数集計
撮影後の「Result」フォルダを確認し、電柱ごとに何枚の画像があるかを集計します。
1. **出力ファイル名**: 保存するファイルの名前を入力します。
2. **参照フォルダ**: 「参照して保存」を押し、画像が格納されている `Result` フォルダを指定します。
3. **実行**: 「処理実行」をクリックすると、フォルダ配下の画像をカウントし、最大4枚までの集計結果をCSVで出力します。

### ③ 緯度経度付URL作成
CSVのリストから、Googleマップで位置を確認するためのURLを一括生成します。
1. **出力ファイル名**: 保存するファイルの名前を入力します。
2. **元ファイル**: 緯度・経度情報が含まれるCSVを選択します。
3. **実行**: 「CSV出力」をクリックすると、Googleマップへのリンクが含まれた新しいCSVが作成されます。

## 3. アップデート方法

### インストーラー版の場合
1. 新しいバージョンの `Choipro_Setup_vX.X.X.exe` をダウンロードします。
2. そのまま実行して上書きインストールしてください。設定は保持されます。

### バイナリ直ダウンロード版の場合
1. 新しいバージョンの `ちょいプロ(仮).exe` をダウンロードします。
2. 古いEXEファイルを削除し、新しいEXEファイルに置き換えてください。
   - データベースは別フォルダ（AppData）にあるため、EXEを入れ替えても設定は消えません。
